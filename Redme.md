# My Developer Portfolio

A modern, responsive personal portfolio website built to showcase my projects, skills, and experience as a  BackEnd developer.

## 🚀 Live Demo
   https://my-portfolio-fawn-psi-58.vercel.app/

## ✨ Features

- **Modern UI/UX**: Built with React and styled using Tailwind CSS v4 for a clean, responsive design.
- **Interactive Elements**: Beautiful icons provided by `lucide-react` and `react-icons`.
- **Contact Form**: Functional contact form powered by EmailJS, allowing visitors to reach out directly.
- **Fast & Optimized**: Scaffolded with Vite for lightning-fast hot module replacement (HMR) and optimized production builds.
- **Fully Responsive**: Looks great on desktops, tablets, and mobile devices.

## 🛠️ Technologies Used

- **Frontend Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Email Service**: [EmailJS](https://www.emailjs.com/)
